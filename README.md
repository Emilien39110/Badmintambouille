# Badmintambouille
Emilien BOITOUZET - Septembre 2024

## Présentation 
Badmintambouille est une application permettant de générer des tournois de badminton en double mixte. A chaque tour il y a un tirage au sort de la paire adversaire mais aussi des partenaires.  
Un classement est caculé à l'issue de chaque match permettant de récompenser la meilleure femme et le meilleur homme.
